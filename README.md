## Dependencies

QARK:
https://github.com/linkedin/qark

Python3:
https://www.python.org/downloads/

lxml:
pip install lxml

Beautifulsoup:
pip install beautifulsoup4

Seaborn:
pip install seaborn

NumPy:
pip install numpy

MatPlotLib:
pip install matplotlib

## Using the program

Create an APKs folder and put your apk files in there.

Executing:
``` python3 experiment.py ```
